# Markdown Note Taker
一个轻量级的 Web 版 Markdown 笔记应用，支持笔记的创建、查看、编辑、删除（CRUD）、关键词搜索，以及 Markdown 实时预览功能。

## 🌟 功能特性
- ✅ Markdown 编辑与实时预览（支持标题、列表、代码块等核心语法）
- ✅ 完整的笔记 CRUD 操作
- ✅ 关键词搜索（匹配标题/内容）
- ✅ 自动记录笔记创建时间和最后更新时间
- ✅ 简洁友好的 UI 界面，支持编辑/预览模式切换

## 📋 环境要求
- Python 3.7 及以上版本
- 网络连接（加载 Marked.js 依赖）

## ⚡ 一键启动指南
### 1. 克隆/下载项目
将项目文件保存到本地，进入项目根目录：
```bash
cd C:\pycharm\pythonProject\markdown-notes  
```

### 2. 安装依赖
执行以下命令安装项目所需依赖：
```bash
pip install -r requirements.txt
```

### 3. 启动服务
```bash
python app.py
```

### 4. 访问应用
打开浏览器，访问以下地址即可使用：
```
http://127.0.0.1:5000
```

## 📖 使用说明
### 基础操作
1. **新建笔记**：点击左侧「新建笔记」按钮，输入标题和 Markdown 内容，点击「保存」即可。
2. **编辑笔记**：点击左侧笔记列表中的笔记项，修改内容后点击「保存」，自动更新最后修改时间。
3. **预览笔记**：点击「预览/编辑」按钮，切换到预览模式查看 Markdown 渲染效果。
4. **搜索笔记**：在左侧搜索框输入关键词，实时筛选匹配的笔记。
5. **删除笔记**：选中要删除的笔记，点击「删除」按钮，确认后即可删除。

### Markdown 支持语法
- 一级标题（`# 标题`）、二级标题（`## 标题`）、三级标题（`### 标题`）
- 无序列表（`- 列表项`）
- 代码块（```python ... ```）

    │   └── style.css     # 样式文件

## ❗ 常见问题
1. **保存笔记无反应**：检查是否进入项目根目录启动服务，确保控制台无报错（F12 查看浏览器控制台）。
2. **数据库字段报错**：删除旧的 `notes.db` 文件，重启服务自动重建数据库。
3. **依赖安装失败**：升级 pip 后重试（`python -m pip install --upgrade pip`）。

  注：因网络环境限制，项目源码以压缩包形式上传，解压后可直接运行，启动步骤见 README.md
